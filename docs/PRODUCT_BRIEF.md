\# Product Brief — Nuzlocke Companion



Nuzlocke Companion is a desktop app that lets players experience Pokémon Nuzlocke runs with a polished visual interface while playing.



The app is designed for the player, not primarily for viewers, streamers, or video editing. The goal is to make a personal Nuzlocke run feel like a professional interactive layout.



\## Main idea

The app shows:

\- the game area,

\- the current team,

\- lives,

\- badges,

\- current route,

\- capture status,

\- current level cap.



\## What the app is not

\- It is not an emulator.

\- It does not include ROMs.

\- It does not modify Pokémon games.

\- It is not a full Nuzlocke database.

\- It is not focused on deaths, box management, items, logs, or timeline events in the MVP.



\## MVP goal

Build a beautiful manual version first. Automation comes later.
